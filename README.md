# BoringTools 🚀

**100 Days. 100 Boring Tools.**

A collection of simple, useful micro-tools built daily to solve everyday problems.

---

## 🛠️ Day 1: Text Formatter

A simple tool to quickly format your text.

### ✨ Features
- Convert text to UPPERCASE
- Convert text to lowercase
- Capitalize each word
- Clean extra spaces
- Copy formatted output
