# BoringTools 🚀

**100 Days. 100 Boring Tools.**

A collection of simple, useful micro-tools built daily to solve everyday problems.

---

## 🛠️ Day 1: Text Formatter

A simple tool to quickly format your text.

### ✨ Features
- Convert text to UPPERCASE
- Convert text to lowercase
- Capitalize each word
- Clean extra spaces
- Copy formatted output

---

## 🛠️ Day 2: JSON Formatter

Format and validate JSON instantly.

### ✨ Features
- Pretty print JSON
- Detect invalid JSON
- Copy formatted output

---

## 🛠️ Day 3: Word Counter

Count words, characters, and sentences instantly.

### ✨ Features
- Word count
- Character count
- Characters without spaces
- Sentence count
